# Welcome to AĒR

**AĒR** (ancient Greek ἀήρ: the lower atmosphere, the surrounding climate) is a modular system for the real-time analysis and long-term observation of societal discourses.

This documentation portal contains the complete [arc42 architecture description](arc42/01_introduction_and_goals.md) of the system.

## Navigation Guide
Please use the menu on the left to navigate through the different layers and Architecture Decision Records (ADRs) of the project. The documentation follows the "Docs-as-Code" principle and is maintained parallel to the source code.
